How the indexer saves tokens

The indexer pre-builds .codebase-indexer/docs/ index files so Claude reads a compact summary instead of loading raw source code. The baseline is what it would cost to answer the same questions by scanning source files directly — a counterfactual estimate based on project size.

Tokens → pages: ~750 tokens ≈ 1 page of text. A 20,000-token saving is roughly 27 pages of source code Claude no longer needs to read.

ModeWhat it does
